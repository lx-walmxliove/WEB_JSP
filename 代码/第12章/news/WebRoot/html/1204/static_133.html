微软正式发布Windows 8消费者预览版

作者：admin    日期：2012-04-17 15:39:00.0    查看次数:0

2月29日消息，微软今天22-24时（当地时间15-17时）在西班牙巴塞罗那2012年世界移动通信大会上发布其下一代操作系统Windows 8消费者预览版（beta测试版），今天共发布“普通版”和“企业版”两个版本。

自今日起，Windows 8消费者预览版可在全球范围内下载，并提供英语、法语、德语、日语和简体中文五个语言版本，下载地址。该版本将于2013年1月16日过期。

与此同时，Windows应用商店（Windows Store）也伴随着Windows 8消费者预览版的发布而正式开放。其中深受网友喜欢的QQ、腾讯浏览器、QQ音乐等应用程序均在首批应用名单中，网友可通过Windows Store免费下载。

Windows 8消费者预览版将被应用在台式机、笔记本电脑和平板电脑等诸多设备上。

在发布会上，微软Windows及Windows Live事业部总裁史蒂文·辛诺夫斯基称Windows 8在Windows 7系统开发完成时就已开始部署。他坦言，尽管Windows 7是微软历史上最成功的操作系统，但时代在变，移动终端大潮来袭，微软也必须顺应潮流，做出适合移动终端和PC为一体的操作系统，为消费者提供更多选择。

辛诺夫斯基强调，微软Windows 8将从硬件、应用软件和开发员体验等三部分着手为消费者提供不妥协的体验。

10万次的修改

辛诺夫斯基同时透露，自2011年9月微软在Windows BUILD大会上首次公开其Windows 8开发者预览版以来，微软已对Windows 8系统进行了超过10万次的修改。

Windows 8开发者预览版自发布以来，下载次数已经超过300万次。

而接下来的日子里，微软还将发布最终发布Windows 8候选版和生产商发布版（Release to Manufacturing）。

戴尔高管和微软高管此前均暗示，Windows 8正式版将于2012年10月推向市场。

Windows 8消费者预览版基本配置需求

Windows 8消费者预览版的几个变化：

  • 全新Logo将出现在Charm栏中，可定义颜色；
  • 任务栏右侧的“显示桌面”按钮将不复存在，不过功能尚存(鼠标移到最右侧悬停显示桌面)；
  • 全新的安装图标和启动画面。

    Windows 8 消费者预览版的一些关键更新：

  • 范围广泛的产品变更和改进；
  • 包含全新应用程序的Windows应用商店；
  • 从基于Windows的PC和Windows Phone 7连接到云；
  • Internet Explorer 10平台预览版5；
  • 预览全新硬件功能。
