前瞻：微软SharePoint 2010新特性预览

作者：admin    日期：2012-04-17 16:14:00.0    查看次数:0
 

2009年第四季度继微软Windows 7、WS 2008 RC2、Exchange 2010等陆续发布或推出Beta版本之后，旗下SharePoint 2010也走进了大家的视线，对于新一代为企业提供内容管理的软件，相比之前的2007版有什么特色呢？下面就为大家带来SharePoint 2010的11大新特性介绍。

　　据Gartner在11月24号报告中称，2010年数据中心的成本将会持续增加，虽然成本问题在近年来一直困扰着数据中心的管理者，但分析师表示在2010年此状况将更加严重。建议企业应该更加主动的面对即将发生的问题，使自身能够持续的发展。

　　微软即将发布的SharePoint 2010系列软件就可以很好的处理这一问题，对于文档资料等信息进行集中系统的管理，包括资源共享和数据的及时更新和统一界面访问等解决方案的提供，首先 就为企业在IT设施和能源上节省了成本，也更加方便人员操控，有效的控制了人员成本的增加以及管理复杂度的繁杂。

　　SharePoint 2010的推出将成为了企业用户解决内容管理和成本增加方面困扰的利器。为微软网络产品和服务汇总项目，它能为各大企业提供网站托管、协作功能、管理模块及文档管理平台等一系列解决方案。

　　此外，SharePoint 2010版还加强了对微软Office办公套件的整合。利用该升级产品，外部开发者和网站设计者可开发、整合更多自家产品和服务(如企业博客及商务应用程序等等)。

　　作为SharePoint 2007的重大升级版，SharePoint 2010版改进并增加了多项功能，如用户界面、在线编辑及支持更多浏览器等等增加在Web上的体验。为企业用户提供了一个功能更加丰富的单一、统一的商业 平台，开发人员可以利用熟悉的工具以及户内平台和云平台上SharePoint核心功能构建出多样性的解决方案。

 

　　SharePoint 2010新特性：

 

　　1.全新的视频预览功能

　　SharePoint 2010包括功能支持包括新媒体和一个Silverlight的内容浏览器Web部件。微软内嵌了Silverlight功能(一种工具，用于创建交互式 Web应用程序)和全站支持AJAX功能，让企业用户很方便地给网站添加丰富的多媒体和互动性体验。

　　我们通过Silverlight Web Part功能，在网页上设置显示一个视频显示框，这是以前没有的功能。企业可以利用这种能力建设自己的视频网站而不需要额外的编程。

 

　　2.更好的备份功能

　　全新的备份和恢复功能，让你能够更加方便的选择需要备份的组件，节省操作时间也缩短了之前复杂繁琐的过程。

 

　　3.系统健康检查功能

　　SharePoint 2010的管理中心网站也经过了重新设计，能够提供更好的可用性，包括检测SharePoint服务器的工作状况这一新功能。

　　4.增加简化的编辑格式

　　SharePoint 2010就像wiki一样简单， 虽然用户可以使用功能丰富的Office来实现编辑功能，但HTML这种标记语言使用起来并不是很方便，它总是需要通过编辑工具生成。而另一方面，Wiki是容易编写的语言代表。

　　SharePoint 2010支持Wiki标记，而SharePoint 2010所见即所得的编辑方式也让操作者能够很好的掌控设计的准确性。在某些方面，SharePoint 2010甚至比许多Wiki更容易使用。

 

　　5.卓越的Excel图表功能

　　通过此功能，Excel Web Access能够让你将实时的Excel图表和数据信息嵌入SharePoint 2010的网页之中，图表内容将监视操作者所有的更新，使得表格保证显示最新数据内容。

 

　　6.强大的ribbon风格整合功能界面

　　微软Office 2010的ribbon风格功能界面也许你记忆犹新，但是一下子SharePoint 2010将众多ribbon风格的功能选项融合起来，让产品更加具有凝聚力，也让操作更加方便、准确。
