.NET Framework 4.5开发者预览版发布！

作者：admin    日期：2012-04-17 15:55:00.0    查看次数:0
 

随着Windows 8和Visual Studio 11开发者预览版的发布，NET Framework 4.5开发者预览版也发布了单行版，目前相关文件包已可在微软下载中心获取。

下载.NET Framework 4.5开发者预览版和4.0正式版：http://www.microsoft.com/download/en/search.aspx?q=.NET%20Framework%204.5

微软.NET标识

与以往惯例相同，开发者可选择在线安装或下载完整离线安装包。在线安装文件不足1MB大小，离线完整版则有近50MB。普通用户可暂时不必升级到4.5版，因为目前使用最新版.NET框架开发的应用程序还极为罕见。

微软表示，.NET 4.5开发者预览版还是一个社区预览版(CTP版本)，因此即便是开发者也不应将其用于正式工作场合，仅仅适用于测试。

目前，普通用户和开发者的最佳选择仍是.NET Framework 4或.NET Framework 3.5。虽然Windows 8中已经集成了.NET 4.5，但此次发布的单行版仍可以运行于Windows 7和Windows Server 2008 R2上，因此开发者可以将其做为一个学习资源加以利用。

据微软介绍，.NET 4.5更新了公共语言运行时(CLR)、.NET类库和Visual Basic和C#语言，并且增强Web服务器数据访问，改进性能、可扩展性以及可靠性和安全性。

关于.NET Framework版本和依赖关系，可参阅微软官方详细介绍。
